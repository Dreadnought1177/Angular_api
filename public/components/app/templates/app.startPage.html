Welcome to my simple admin dashboard. Built with some simple Bootstrap, this project is designed to illisturate my ability to design a Angular front-end that can efficently use and minipulate data consumed by my own custom RESTful API. The project allows you to add, edit and remove user's and user profile information and includes a message board complete with comments. Click the start button to create your own profile or use the login in the top right corner to get started

AngularJS

I love Angular and chose to use it as my front-end. I used Angular-resource and Angular-ui-router to help me retrieve and control the data from my API. I used Express as my web appilication server framework.

MongDB with MongoEngine

The API is built using Django and features MongoDB as my NoSQL database. After a bit of a learning curve, I found Django-MongoDb to be my ideal backend solution.

Combining these two technologies I was able to build clean and flexible models and Pythonic serializers. Django Using MongoDb also eliminated the need for SQL database migrations.
